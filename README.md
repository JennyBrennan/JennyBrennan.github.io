# jennybrennan.com

Blog and personal website, built using Gatsby, based on the Gatsby starter blog, at `https://github.com/gatsbyjs/gatsby-starter-blog`.

## Running in development
`gatsby develop`

## Deploying to gh-pages
`npm run deploy`
